# Programação em Pascal

Repositório com códigos em Pascal desenvolvidos durante a disciplina de Estrutura de Dados, cursada na Fatec-SP, ministrada pelo professor Victor Antonio Troitiño, durante o 2º semestre de 2025, no período noturno.

O foco da disciplina é o estudo dos diferentes tipos de Estruturas de Dados (pricipalmente listas, pilhas, listas encadeadas e os algoritmos relacionados à essas estruturas) e, não a linguagem de programação Pascal. Mas, para mim, assim como aprender um novo idioma, sempre é empolgante aprender uma nova linguagem de programação.

Procurei pesquisar sobre as boas práticas do desenvolvimento em Pascal, suas convenções, a estrutura dos programas etc. Por questões de estética, preferi adotar os padrões do Pascal moderno (em detrimento ao acadêmico).

Os programas estão muito carregados pois, por questões de saúde, estou fazendo regime domiciliar neste semestre (2º semestre de 2025). Então, os estudos estão fora do padrão, pelo menos na primeira parte do semestre.
